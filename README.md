# Data Engineering Practical Assignment

## Overview

This project demonstrates Linux administration, PostgreSQL database management, networking, SCP file transfers, and documentation skills required in Data Engineering.

---

## Environment

- Ubuntu 24.04 LTS
- PostgreSQL 16
- DBeaver
- GitHub

`whelman`
